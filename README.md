# Sabrine
 Basing our future multibillion dollar company on Sean Parker and Zucc
 
 ![Sean, Napster, Parker](Sean.png)

## NodeJS
### Dependencies
Simply run
```
npm install
```
### To start server
Run in root folder
```
DEBUG=myapp:* npm start
```
## Flask

### Dependencies
Simply run
```
pip install -r requirements.txt
```

### To start Flask server
Run in root folder
```
flask --app main.py run
```
